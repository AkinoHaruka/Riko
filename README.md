# RIKO — AI Chat App

跨平台 AI 聊天桌面/移动应用，基于 DeepSeek AI。

## 下载

前往 [Releases](https://github.com/AkinoHaruka/Riko/releases) 下载最新 APK。

## 平台支持

- **Android**: APK 直接安装
- **Windows / macOS / Linux**: 桌面端（Flutter 构建）
- **iOS / Web**: 计划中

## 技术栈

- 前端: Flutter 3.x (Dart)
- 后端: Fastify 5 + TypeScript
- AI: DeepSeek (deepseek-v4-flash / deepseek-v4-pro)
- 数据库: SQLite (better-sqlite3 / sql.js)

## 构建

```bash
# Android
flutter build apk --release

# Windows
flutter build windows --release
```
